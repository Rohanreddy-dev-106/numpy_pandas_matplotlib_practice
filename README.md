# 🐍 Python Data Science Basics

A beginner-friendly repository covering **NumPy**, **Pandas**, and **Matplotlib** with practical examples.

---

## 📦 Libraries Covered

### 🔢 NumPy (`basic.py`, `basic02.py`)
- Array creation and reshaping
- Mathematical operations
- Advanced indexing and slicing
- Broadcasting

### 🐼 Pandas (`pandas_basics.py`)
- Creating and inspecting DataFrames
- Selecting, filtering, and sorting data
- Handling missing values
- GroupBy and aggregation
- Merging and pivot tables
- Reading/writing CSV files

### 📊 Matplotlib (`matplotlib_basics.py`)
- Line charts, bar charts, histograms
- Scatter plots and pie charts
- Customizing titles, labels, and legends
- Saving plots as image files

---

## 🚀 Getting Started

### Prerequisites
Make sure you have Python 3.8+ installed.

### Installation

```bash
# Clone the repository
git clone https://github.com/Rohanreddy-dev-106/<repo-name>.git
cd <repo-name>

# Install dependencies
pip install numpy pandas matplotlib
```

### Run any file

```bash
python basic.py
python basic02.py
python pandas_basics.py
python matplotlib_basics.py
```
